Get summary stats

Calculate plus minus per name (i.e., player) calculated as

\[ plus minus = (goals + Ds) - (throwaways + drops) \]

Arrange names in descending order.

plus_minus <- bh %>%
  select(
    name, action
  ) %>%
  group_by(
    name
  ) %>%
  summarise(
    goals = sum(action == 'Goal'),
    Ds = sum(action == 'D'),
    throwaways = sum(action == 'Throwaway'),
    drops = sum(action == 'Drop'),
    p_m = (goals + Ds) - (throwaways + drops)
  ) %>%
  arrange(desc(
    p_m
  ))

Take out the player Anonymous [he is probably a scary hacker]

mvplus_minus <- plus_minus %>% 
  filter(name != "Anonymous")

Plot


A plot approximating the Wildfire MVP by plotting season Ds vs.season goals.

mvp_plot_2 <- ggplot(data = mvplus_minus, aes(x = Ds, y = goals, 
                                   label = name, angle = 45))
mvp_plot_2 + geom_point() +
  ggtitle("MVP") +
  xlab("Season Blocks") + 
  ylab("Season Goals") +
  geom_text_repel(aes(label = name), 
                  box.padding = unit(0.45, "lines"))


(Of course, this plot doesnt consider the bad things [drops and throwaways], so the p_m column of the mvplus_minus tibble is probably a more accurate indicator of a players contributions.)
